Materialize Accordions Pro allows you to add a simple accordion with titles and body text via a shortcode; with more options


== Description ==

This plugin allows you to add a simple accordion with a title and body text via a shortcode with options on the sidebar. The inteface is really easy to use and you can sort sections around with the Visual and Text editor intact.

This plugin works with the Classic editor or the Gutenberg shortcode block and should have no conflicts with other plugins.

== Installation ==

1. Upload the materialize_accordions.zip folder to the '/wp-content/plugins/' directory.
2. Activate the "accordion-and-accordion-slider" list plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode in post or page with XX being your post/page ID
<code> [MA_shortcode id=XX] </code>


== Frequently Asked Questions ==

Q- Can it be used directly in a template?

A- Yes, just add this to your template file: `<?php echo do_shortcode('[MA_shortcode id=XX]'); ?>` and replace [MA_shortcode id=XX] with your actual shortcode.
